初Git

(ja) はじめてのpush

初めてGithubにpushなるものをしました。 加えてGitpagesでのWebページの立て方も教えていただき、hugoを使ったblogを今書いています。

Gitへpushするとは話に聞いてはいましたが、実際やったことがなかったので新鮮でした。
当初CUIに日和ってGitHub Desktop を使っていましたが、途中からウィンドウの移動がめんどくさくなってGit Bashに回帰しました。

コマンドライン最高！

＜＜＜自分へ＞＞＞draft:trueは下書きです。

以下Git bushで使うコマンドの備忘録です。

$ git status
#なにはともあれstatus

git add <ステージにあげたいやつ>
#最終的にpushしたいやつをなんかあげる

git commit
#push前にコメントつける

git push origin <brancnを指定>
#pushだ！！！！！！！

git remote -v
#origin調べるらしい
